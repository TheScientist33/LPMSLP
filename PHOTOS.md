# Photos du site — où apparaît quoi

> **Pour remplacer une photo :** dépose ton nouveau fichier dans le dossier
> indiqué avec **exactement le même nom** (ex. remplacer `piscine-hero.jpg` par
> une autre photo nommée `piscine-hero.jpg`). Sur GitHub : ouvre le dossier →
> « Add file » → « Upload files », ou clique le fichier puis l'icône crayon.
> Le site ré-optimise et met à jour automatiquement au déploiement.
>
> Formats acceptés : `.jpg`, `.jpeg`, `.png`, `.webp`. Privilégie des photos
> lumineuses et horizontales (paysage), au moins 1600 px de large.

Les photos sont dans `src/assets/images/`.
